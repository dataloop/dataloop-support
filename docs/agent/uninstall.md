## Uninstall


On Linux you can simply remove the `dataloop-agent` package.

### Redhat based:

```
yum remove dataloop-agent
```

- - -

### Debian based:

```
apt-get remove dataloop-agent
```
- - -

### Windows

On Windows you can remove via add/remove programs.
